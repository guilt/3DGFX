# Directory Layout

- `EXAMPLES/` - Pre-built executables + data files (.BMP, .MSH, .TEX, .PAL, .IM)
- `EXMPLSRC/` - Source code for all 12 example programs
- `INCLUDE/` - Library headers (core headers + FTARK/FUNTI per-range font data)
- `DOC/` - Documentation
- `BGI/` - BGI driver binary (SVGA256.BGI)
